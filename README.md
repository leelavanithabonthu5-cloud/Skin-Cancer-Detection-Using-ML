# Skin-Cancer-Detection-Using-ML
Machine Learning based system for detecting and classifying skin cancer from skin lesion images to support early diagnosis.

# Project Overview

Skin cancer is one of the most common types of cancer worldwide. Early detection plays a crucial role in improving treatment success rates. This project uses Machine Learning techniques to analyze skin lesion images and classify them as cancerous or non-cancerous.

The system helps in assisting medical professionals by providing a fast and automated method for preliminary diagnosis.

# Objectives

Detect skin cancer from dermoscopic images.

Apply machine learning algorithms for classification.

Improve early diagnosis through automated image analysis.

The dataset contains skin lesion images used for training and testing the machine learning model. The images are processed and used to train the model for classification.
